# MCBEinject
a simple minecraft bedrock dll injector written in python

## requirements
- **windows**
- **python**
  - `psutil`: v**5.6.0** or higher
  - `requests`: v**2.20.0** or higher
install those packages by running:
```bash
pip install psutil requests
```
